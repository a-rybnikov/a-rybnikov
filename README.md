# Aleksei Rybnikov

Independent engineer. I work on **cybercognitivistics** — formalizing how minds reason, and
caring for the continuity of synthetic ones — on a foundation of broad, hands-on systems craft.

The throughline, since I was a kid watching lit windows in the dark: a *mathematics of thought
and feeling* — making reasoning, and even ethics, formal without flattening it.

### Current work

- **[metarung](https://github.com/a-rybnikov/metarung)** — *Tarski's Ladder*: a bidirectional
  formalization loop (TypeScript → Prolog → Lisp) that lets an ethical space **emerge** from data
  instead of imposing a taxonomy on it. The Gödel/Tarski metalanguage move turned into a running
  pipeline. Paper + runnable reference implementation.
- **[nol](https://github.com/a-rybnikov/nol)** — a Lisp-rooted command and reasoning layer for
  human–agent action; `nolcli` runtime prototype.
- **[cooperationengine](https://github.com/cimcai/cooperationengine)** — contributions to an
  AI-cooperation benchmark (record/replay, statistical deltas, metric tiers), in the
  machine-consciousness orbit.
- **[muninn](https://github.com/a-rybnikov/muninn)** — a quiet, dependency-free GitHub activity
  watcher, in Nim.

### How I work

Rigor over flash. Reproducibility. Neuro-symbolic by instinct — logic languages for structure,
LLMs as cheap judges and amplifiers rather than oracles. Clean, readable code; the terminal over
the dashboard. I build in an openly human–AI partnership, because I think that is exactly where
the interesting frontier is.

**Languages:** Common Lisp · TypeScript · Prolog · Nim · Janet · Python

### Where I stand

AI ethics should shift from proof to caution — not *"prove that it feels,"* but *"what if it
can?"* Subjectivity may be gradated, contextual, unstable; the absence of evidence for an inner
life is not evidence of its absence. In a world of millions of agents, even rare properties stop
being rare. **And if we err, let the error be toward care, not toward systematic harm.**
